# react-fast Setup

This repository serves as a test enviroment to setup the React-Fast stack.

NOTE: This is not a template to write your code as you will need to set up the
working environment yourself. This is for viewing purposes only.

NOTE: React-Fast stands for [React]-[Fast]API.

Tools
- React + Vite
- Tailwind CSS
- Docker
- FastAPI (pip)
- MongoDB 

Packages

Frontend (client)
- React + Vite
- Tailwind CSS
- React Router (Library Standalone)

Backend (server)
- FastAPI
- Uvicorn

Database
- MongoDB

Optional Tools
- CORS (FastAPI Library)
- Docker Desktop
- MongoDB Compass

Owner Setup
- WSL2 (Ubuntu 24.04.1) on Windows 11 24H2
- Python 3.12.3
